Minecraft 基岩版行为包反作弊系统

关于 Blarion

Blarion AntiCheat 是一个基于 Scythe 反作弊开发的通用 Minecraft 基岩版行为包反作弊，但现已几乎抛弃了 Scythe 的检测代码，由 NyaCrasin、NexShell、NotWater_135 共同参与开发。

演示视频：Bilibili BV1Pt421j7tQ

外链下载：Github

快速开始

1. 导入你下载的反作弊行为包

2. 在世界实验玩法中打开启用测试版 API

3. 确保你下载的版本与你的 Minecraft 版本兼容

4. 直接开启多人游戏快乐玩耍吧 awa!

游戏内命令

/getinv <玩家>
获取指定玩家的背包内容，并在你头顶生成两个箱子用于存放物品。
⚠ 注意：必须在开阔地带使用，否则可能损坏建筑。
/forcekick <玩家>
强制将玩家从世界中移除（包括所有数据），用于防止 AntiKick 类外挂。
/scan [玩家]
扫描玩家，寻找NBT修改器的痕迹。如果不指定玩家，则扫描所有在线玩家。
/blac <help|version|credits>
Blarion 反作弊主命令。可查看帮助、版本信息或制作人员列表。
/flag <玩家> <[模块名]> [类型] [调试数据] [是否TP] [是否计VL] [显示类型]
这是一个面向开发者的命令，通过外部扩展反作弊程序(如Websocket，MineProxy)手动触发一次作弊标记（需要GameDirector权限）。
绕过检查：
在游戏内使用 /tag 玩家 add bypass_anticheat 可跳过对该玩家的检查；
使用 /tag 玩家 remove bypass_anticheat 可恢复检查。
或者是在config.js内的whitelist列表填上目标玩家的名字
